# 超市仓库管理系统
在中国软件杯，老师让做一个超市仓库管理系统
# 搭建框架
采用后端的技术是 Java Web， Spring MVC + Spring + Spring JDBC

前端采用的技术是 JQuery ,BootStrap ，由我的队员@coolfeel 编写完成
